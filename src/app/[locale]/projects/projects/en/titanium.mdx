## Overview

In e-commerce, flexibility in management and ease of use are key success factors. With this in mind, we built Titanium—a complete digital ecosystem for a leading electronics store in Tripoli, Libya. The goal was clear: build an integrated system that serves the customer, driver, and management in a unified, modern experience suited to the store's scale and reach.

## What Was Delivered

### Customer Mobile App

A dedicated mobile app with a modern, smooth interface that displays all products by branch, with full Arabic and English support.

### Super Admin Dashboard

A central dashboard for managing products, distributing them across branches, tracking inventory, and monitoring orders.

### Branch Dashboards

A separate dashboard for each branch to manage orders and inventory independently.

### Driver App

A dedicated app for drivers to receive orders, track customer locations, and update order status in real time.

### Authentication

Secure, fast sign-in via Firebase Authentication with Google and Apple support.

### Flutter UI

A responsive, modern interface built with Flutter, compatible with Android and iOS.

### Laravel Backend

A robust Laravel backend that connects all project components seamlessly.

### Hosting

The project is hosted on a high-performance VPS from Libyan Spider for stability and speed.

## Outcome

Titanium reflects a real-world model for building a complete e-commerce system in Libya, using modern technologies and a user experience that meets local market needs. The system connects customers, drivers, and management in a single, efficient workflow.
